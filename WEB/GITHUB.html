GITHUB

GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones Git. Se utiliza principalmente para la creación de código fuente de programas de ordenador. El software que opera GitHub fue escrito en Ruby on Rails. Desde enero de 2010, GitHub opera bajo el nombre de GitHub, Inc.

Tras esta pequeña definición de que es GitHub os diré cuales son mis planes aquí ;). En esta pestaña os enseñaré a usar la plataforma GitHub. Esta es una plataforma que se encuentra en inglés, al principio puede resultar muy liosa, pero céntrate solo en las funciones que sepas usar o te pidan, ya que en el mundo de la programación tenemos infinitas posibilidades además de las que nos aporta la propia plataforma.

Github incluye una forma muy buena de aprovechar el trabajo de otras personas, ya que la gente comparte su trabajo con todo el mundo a través de internet, así mismo puedes coger el trabajo de otras personas y modificarlo a tu antojo, o incluso reciclar códigos que la gente desecha. Además, por si no lo sabías es una plataforma que también te permite hacer algoritmos y compartirlos con todo el mundo :).

Mediante fork puedes clonar, dentro de tu espacio, el proyecto de otra persona. La ventaja principal es que nadie tiene que dar autorización para la copia y, cualquier cambio que hagas en el código no afectará al archivo original, de eta manera ningún archivo se corrompe y lo único que habrás hecho ha sido copiar el código de una persona, y crear sobre él tus modificaciones.

GitHub es una herramienta muy útil si nuestro objetivo es crear un proyecto en conjunto con compañeros y que cada uno pueda ir adjuntando sus avances en el código fuente correspondiente. Para cuando todo el mundo no tenga nada más que aportar tengamos un resultado final “perfecto”.

A continuación como de costumbre os dejo una imagen, en ella os mostraré mi cuenta para que veáis que estoy dado de alta en esta plataforma.

GitHub también podríamos usarlo para hacer trabajos de programación junto a otros desarrolladores, ya que puedes alojar a otras personas en tu poryecto y escribir todos en tiempo real aportando vuestro trabajo e ideas simultaneamente.

Intentad explotar al máximo las posibilidades de esta plataforma, aprended y lo más importante, disfrutad usándola. :)
